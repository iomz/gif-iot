# GIF-IoT
*Device and SensorTag management portal for GIF 2015 Workshop*

## Synopsis
Start GIF-IoT sinatra app:

```sh
% git pull https://github.com/iomz/gif-iot.git
% cd gif-iot && bundle install --path vendor
% foreman start
```
